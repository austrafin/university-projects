1 Starting the Program

When you first start the program you are prompted to enter the initial number of fans. Using the spinner provided, enter this number and press 'START NEW' button. This will create an instance of a fan group with the specified number of fans added to the fan display panel.



If you wish to exit the program instead, press the 'EXIT' button, or 'X' button in the top right hand corner.



If you wish to load a fan setup from a file, press the 'OPEN FILE' button.


2 Fan Operations

2.1 Adding Fans

Fans can be added by pressing 'ADD' button underneath the fan display. The number of fans to be added can be specified by using the spinner next to it. The maximum number of fans in one program instance is 9, and the minimum is 1.




2.2 Removing Fans

Individual fans can be removed by pressing the 'REMOVE' button in the individual controls panel. This removes the fan in the fan display panel as well as its individual control panel.




2.3 Turning Fans On and Off

All of the fans can be turned on and off at the same time by pressing 'ON' and 'OFF' buttons in the 'Group Controls' panel. Individual fans can be turned on and off by pressing the 'ON' and 'OFF' buttons in their dedicated control panels in the 'Individual Fan Controls' panel.




2.4 Reversing Fans

All of the fans can be turned reversed at the same time by pressing 'REVERSE' button in the 'Group Controls' panel. Individual fans can be reversed by pressing the 'REVERSE' button in their dedicated control panels in the 'Individual Fan Controls' panel.




2.5 Changing the Speed of Fans

The same speed for all the fans can be set by adjusting the speed slider in the 'Group Controls' panel. The speed for individual fans can be set by adjusting the speed slider in their dedicated control panel in the 'Individual Fan Controls' panel.




2.6 Changing the Theme of Fans

A theme with distinct colours can be set for each fan. Themes for all the fans can be set by selecting an option from the group of radio buttons in the 'Group Controls' panel. A theme for an individual fan can be set from the radio buttons in the 'Individual Fan Controls' panel.




3 File Menu Options

3.1 Adding a New Fan Group Instance

A new fan group instance can be added by selecting 'File' from the top menu bar and then selecting 'New'. This opens a new window (instance) of the program interface. For instructions on how to proceed from this point, refer to Section 1.




3.2 Saving a Fan Group Instance

A fan group instance can be saved so that it can be opened later. To do this, select 'File' from top menu bar, and then 'Save'.




A save dialog will then appear and you can navigate to a directory where you want to save the file. The file extension for the saved files is '.fan'. Press 'Save' button to save the file and 'Cancel' button if you want to cancel the operation.




3.3 Opening a Saved Fan Group Instance

To load a saved fan group instance, select 'File' from the top menu bar and then 'Open'.



An 'Open' dialog will then appear and you can navigate to the directory where the file is saved and press 'Open' button to load the file. Press 'Open' button to open the selected file and 'Cancel' button if you want to cancel the operation.



3.4 Setting the Look & Feel of the Program

To set the look & feel of the program, select 'File' from the top bar menu and then 'Look & Feel'. A list with different look & feel options is then presented. When an option is selected, the program will be updated with that look and feel.




3.5 Exiting the Program

To exit the program, you can either select 'File' from the top bar menu and press 'Exit', or press 'X' in the top right corner.



You are then asked whether you want to save the fan group instance to a file or not. By pressing 'CANCEL', the program will continue to run and does not terminate. By pressing 'YES' you are presented with a save dialog. For instructions on how to proceed from that point, refer to Section 3.2.
