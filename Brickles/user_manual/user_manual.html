1 Starting a new game

When the program is started, a new game can be started by pressing the "START NEW GAME" button in the main menu. Alternatively, a new game can be started from the game menu on the top left corner of the game field.


2 Game modes

When a new game is started, a type of game can be chosen from three options below by selecting it from the drop-down list.


2.1 Classic

This is the classic brickles game mode where the bat is on the bottom of the screen and the aim is to break all of the bricks on the top of the screen. Red bricks score you 10 points, yellow bricks 5, green bricks 2 points, and blue bricks 1 point. After breaking all the bricks the game field is reset but the difficulty has incresed!


2.2 Ping Pong

This game is a tennis type of game where two bats are playing against each other. When one player misses, the other player scores. The outcome of the game can be either either player winning or a tie. The game ends when the rounds reach zero.


2.3 All In

In this game the aim is to break all of the bricks in the middle of the game field by bouncing the ball off the bats on all four sides of the game field. Red bricks score you 10 points, yellow bricks 5, green bricks 2 points, and blue bricks 1 point. After breaking all the bricks the game field is reset but the difficulty has incresed!


3 Loading a saved game

A saved game can be loaded by pressing the "LOAD GAME" button on the Main Menu or alternatively pressing the load game menu item in the game menu on the top left corner of the game field. A file browser dialog will appear and files with the extension '.brick' can be selected from the disk.


4 Saving a game

Current game can be saved by pressing 'Save' button on the top left hand corner of the game menu. A file browser dialog will appear and the brick file can be saved on disk.


5 Setting the look and feel of the program

Look and feel can be set bu pressing the button 'SET LOOK AND FEEL' on the Main Menu. A new dialog window will appear and the look and feel can be selected from the drop-down menu. Alternatively the look and feel can be set from the game menu on the top left corner of the game field.


6 Hall of Fame

The Hall of Fame stores the highest scores scored in the game. In order to make it to the Hall of Fame, the player must score more points than the worst score in the list. When the player makes it to the Hall of Fame, they are prompted for their name. If they do not specify a name "Anon" will be recorded. The Hall of Fame can be accessed by pressing the 'Hall of Fame' button in the Main Menu. Alternatively it can be accessed from the game menu on the top left corner of the game field.


7 Themes

The game offers three themes: Default, Meadow and Royal. They define the colour of the bat, walls and ball. Themes can be set from the game menu on the top left corner of the game field.


8 Exiting the game

The game can be exited by pressing the 'Exit' button in the Main Menu or alternatively selecting 'Exit' from the game menu on the top left corner of the game field. If currently playing a game the user can also exit to Main Menu by selecting 'Exit to Main Menu' from the game menu.
